Numbers Which Sum of Powers of Its Digits Is The Same Number.

The sum of the third-power of each of its digits is equal to 153.

This is my solution of this task. It took a lot of time to solve, because i couldn't find what i did wrong.

function eqSumPowdig(hMax, exp) {
   let arr;
   let result = [];
   let n = 0;

    for (j = 2; j <= hMax; j++) {
	  j = String(j); 
	  arr = Array.from(j);
										
	    for (i = 0; i < arr.length; ++i) {
		   n += arr[i]**exp;
		} 
		if (n == j)  {
		  result.push(n)
		  n = 0
		} else { n = 0 }
	  }      
  return(result);
}
